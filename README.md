# Análisis de Supervivencia del Titanic

Este es mi primer proyecto de análisis de datos, enfocado en explorar el famoso conjunto de datos del Titanic.

##  Objetivo

El objetivo de este proyecto es usar Python, Pandas y Seaborn para analizar los datos de los pasajeros y responder a dos preguntas clave:

1.  ¿Cómo afectó el **género** a la probabilidad de supervivencia?
2.  ¿Cómo afectó la **clase social** (clase del ticket) a la probabilidad de supervivencia?

## Herramientas Utilizadas

* **Python 3**
* **Pandas** (para la carga, limpieza y análisis de datos)
* **Matplotlib** y **Seaborn** (para la visualización de datos)
* **Jupyter Lab** (como entorno de desarrollo)

## Hallazgos Clave

Tras analizar los datos, los hallazgos más importantes fueron:

* **Género:** Hubo una diferencia drástica. Las **mujeres tuvieron un 74.2%** de probabilidad de sobrevivir, mientras que los **hombres solo tuvieron un 18.9%**.
* **Clase Social:** El dinero fue un factor decisivo. Los pasajeros de **1ª clase tuvieron un 63%** de supervivencia, comparado con solo un **24%** de los pasajeros de 3ª clase.

## Conclusión

Este análisis básico confirma con datos las conocidas historias sobre el desastre. Demuestra el poder de Pandas para agrupar datos y de Seaborn para visualizar estos patrones de forma clara.
